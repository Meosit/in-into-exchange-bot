Список доступних API з курсами валют:

__Національний банк Республіки Білорусь__
- Сайт: http://www.nbrb.by/
- Позначення: `NBRB`, `НБРБ`
- Базова валюта: `BYN`
- НЕ підтримувані валюти: `ILS`, `BTC`, `ETH`

__Національний банк України__

- Позначення: `NBU`, `НБУ`
